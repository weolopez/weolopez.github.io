‚öôÔ∏è Configuration Options

Basic Usage:

<camera-mouse-status></camera-mouse-status>

With Debug Panel:

<camera-mouse-status show-debug="true"></camera-mouse-status>

Light Theme, Bottom Position:

<camera-mouse-status theme="light" position="bottom"></camera-mouse-status>

No Virtual Cursor:

<camera-mouse-status show-cursor="false"></camera-mouse-status>

Available Attributes:

  • show-cursor="true|false" - Show virtual cursor (default: true)
  • show-debug="true|false" - Show debug panel (default: false)
  • position="top|bottom" - Status bar position (default: top)
  • theme="dark|light" - Status bar theme (default: dark)

üîå JavaScript API

The component also provides a JavaScript API for programmatic control:

// Get component reference const statusComponent = document.querySelector('camera-mouse-status'); // Access the camera service const service = statusComponent.getService(); // Check tracking state const isTracking = statusComponent.getTrackingState(); // Hard refresh the service await statusComponent.hardRefresh(); // Listen to component events statusComponent.addEventListener('tracking-started', (e) => { console.log('Tracking started!', e.detail.service); }); statusComponent.addEventListener('tracking-stopped', (e) => { console.log('Tracking stopped!'); });
